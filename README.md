## Como executar o projeto

### Pré-requisitos
Docker em execução

```bash
# clonar repositório
git clone https://github.com/LuisPaulo1/parking-control-api.git

# entrar na pasta do projeto parking-control-api
cd parking-control-api

# executar o projeto
docker-compose up
```
